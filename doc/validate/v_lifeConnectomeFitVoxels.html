PURPOSE

Illustrate how to open up data and run a small linear fascicle evaluation

function fe = v_lifeConnectomeFitVoxels

DESCRIPTION 

 Illustrate how to open up data and run a small linear fascicle evaluation
 (LIFE) by fitting the fibers with one weight per voxel instead of a
