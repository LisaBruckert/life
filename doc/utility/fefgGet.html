fefgGet

PURPOSE

Get values from a fiber group structure

SYNOPSIS

function val = fefgGet(fg,param,varargin)

DESCRIPTION 

Get values from a fiber group structure

  val = fefgGet(fg,param,varargin)



 Copyright Franco Pestilli (2013) Vistasoft Stanford University.
