mctRunOrigFit

PURPOSE

SYNOPSIS

function fe = mctRunOrigFit(fe,dwi)

DESCRIPTION 

 fe = mctRunOrigFit(fe,dwi)

 Performs a run of the orginal connectome model.

 The original model is a model that assigns the same weight to all the
 fibers in the connectome.
