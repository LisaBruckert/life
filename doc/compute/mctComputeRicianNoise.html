 function n = mctComputeRiceNoise(mu,sd,siz)

 Generates Rician distributed noise.
 
 Rician noise can be added to the diffusion signal to generate simulations.

 Example:
     n = mctComputeRiceNoise(500,10,[10,10]);

0019 % Matlab does not provide a Rician Noise generator. To approximate rician
0020 % noise I add the square of two gaussian distributions with the same
0021 % standard deviation (sd), one with zero mean the other with some mean
0022 % (mu). The square root operation takes me back to the original space.
0023 %
0024 % See "Related Distribution" here:

0028 % generate using the square root of the sum of the squares of two gaussians
0029 x = mu + sd .* randn(siz);
0030 y =      sd .* randn(siz);
0031 n = sqrt(x.^2 + y.^2);
